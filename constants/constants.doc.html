Module Constants

     Version: v2.0
     Date:    October 5, 1999
     Contact: Bruce Wyman

OVERVIEW


     Defines useful constants for Earth in mks units.

     Constants are defined as real parameters.

OTHER MODULES USED


     None.

PUBLIC INTERFACE


  use constants_mod [, only: radius, omega, grav, rdgas, kappa, cp, p00,
                             rvgas, d622, d378, d608, dens_h2o, hlv, hlf, hls, tfreeze,
                             stefan, vonkarm ]

  See the description of public data below.

NOTES ON USAGE:

1) All constants have been declared as type REAL, PARAMETER.

2) Constants have been declared as type parameter and therefore must be used accordingly.
   The value a constant can not be changed in a users program. New constants can be
   defined in terms of values from the constants module using a parameter statement.

       real, parameter [, private]  :: grav_inv = 1.0 / grav

3) The name given to a particular constant may be changed, for example:

       use constants_mod, only:  grav_new => grav

4) Constants can be used on the right side on an assignment statement
   (their value can not be reassigned).

       geopotential(:,:) = height(:,:) * grav

CHANGE HISTORY


changes for v2.0 (10/5/99)

     real, parameter :: d622 = rdgas/rvgas
     real, parameter :: d378 = 1.0-d622
     real, parameter :: d608 = (rvgas-rdgas)/rdgas  ! (or d378/d622)
